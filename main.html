MS-DOS 1.0

Microsoft Disk Operating System 1.0 (Commonly abbreviated as MS-DOS 1.0)
was the first Operating System Microsoft made
It was created on 8-12-91 (12, August 1991)
MS-DOS 1.0 has these commands:
  • FORMAT <DRIVE>
    • Formats a drive
    e.g: FORMAT A:
  • DEL <FILENAME.EXT>
    • Deletes a file
    e.g: DEL IO.SYS
  • REN <FILENAME.EXT> <NEWFILE.TXT>
    • Renames a file
    e.g: REN COMMAND.COM IO.SYS
  • COPY CON <FILENAME.EXT>
    • Creates a file
    e.g: COPY CON MSDOS.SYS
  • DIR
    • Lists files in the current directory
  • DIR *.EXT
    • Lists files with a specific extension
    e.g: DIR *.SYS
  • TYPE <FILENAME.EXT>
    • Returns the contents of the file
    e.g: TYPE COMMAND.COM
